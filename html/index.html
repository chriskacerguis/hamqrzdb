Features

            
Everything you need for amateur radio callsign lookups

        

        

            
            

                

                    
                

                
Blazing Fast

                
Average response time of ~2ms with throughput of 2,500 requests per second

            


            
            

                

                    
                

                
HamDB Compatible

                
Drop-in replacement for HamDB with full API compatibility

            


            
            

                

                    
                

                
Self-Hosted

                
Complete control over your data with Docker deployment

            


            
            

                

                    
                

                
Location Data

                
Includes coordinates and Maidenhead grid squares

            


            
            

                

                    
                

                
Daily Updates

                
Automated updates from FCC ULS database in ~30 seconds

            


            
            

                

                    
                

                
Case Insensitive

                
Works with any callsign case (KJ5DJC = kj5djc)

            

        

    


    
    

        

            

                
API Documentation

                
                
                

                    
Endpoint

                    

                        GET /v1/{callsign}/json/{appname}
                    

                


                
                

                    
Parameters

                    

                        

                            
                                

                                    Parameter
                                    Type
                                    Description
                                

                            
                            
                                

                                    callsign
                                    string
                                    Amateur radio callsign (case-insensitive)
                                

                                

                                    appname
                                    string
                                    Your application name (for compatibility)
                                

                            
                        

                    

                


                
                

                    
Example Request

                    

                        
curl http://lookup.kj5djc.com/v1/KJ5DJC/json/myapp
